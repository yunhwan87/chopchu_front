# 작업 로그 (2026-03-06)

오늘 진행된 주요 업데이트 및 기능 개선 사항입니다.

## 1. 일정 및 프로젝트 관리 기능 강화
- **프로젝트 기간별 일정 필터링**: 선택된 프로젝트의 시작일과 종료일 범위 내에 있는 공식 일정만 촬영 일정 보드에 표시되도록 개선했습니다.
- **일차별 필터 기능**: 촬영 일정 보드 상단에 '1일차', '2일차' 등 프로젝트 진행 일차별로 일정을 필터링할 수 있는 탭을 추가했습니다.
- **D-Day 표시**: 프로젝트 시작일까지 남은 기간을 계산하여 상세 화면 상단에 D-Day 배지로 표시하는 기능을 구현했습니다.
- **일정 취소 요청**: 각 일정 카드에 '취소요청' 버튼을 추가하여 긴급 상황 시 관리자에게 알림을 보낼 수 있는 UI/UX를 구축했습니다.

## 2. 대시보드 및 정보 시각화 개선
- **프로젝트 요약 정보**: 홈 화면의 프로젝트 카드 및 상세 화면에서 '총 진행 일수'와 '총 참여 인원'을 직관적으로 확인할 수 있도록 배지 형태의 UI를 적용했습니다.
- **내비게이션 동기화**: 홈 화면에서 특정 프로젝트를 선택하면 상단 탭이 자동으로 '메인'으로 전환되며 해당 프로젝트의 상세 정보가 바로 펼쳐지도록 연결 로직을 강화했습니다.

## 3. UI/UX 레이아웃 최적화 (가장 최근 작업)
- **하단 탭바 겹침 현상 해결**: 모든 주요 화면(홈, 메인, 섭외, 요청)에서 리스트의 마지막 항목이 하단 고정 탭바에 가려지지 않도록 하단 여백(`paddingBottom`)을 최적화했습니다.
    - `LocationManager.js` (섭외 메뉴)
    - `ScheduleScreen.js` (메인 메뉴)
    - `CommunicationScreen.js` (요청 메뉴)
- **전체적인 일관성 유지**: 모든 화면에 동일한 하단 여백 기준(120px)을 적용하여 스크롤 시 끝까지 쾌적하게 정보를 확인할 수 있도록 조치했습니다.
